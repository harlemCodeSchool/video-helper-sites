Web Basics

Web Browsers, IE:

  • Chrome
  • Safari
  • Internet Explorer
  • Firefox
  • Opera

All are designed to do one thing:

Render HTML, CSS and Javascript

HTML

Hyper Text Markup Language is where you write the content of your page: this includes the text on the page and images and more.

CSS

Cascading Style Sheets are the layout and coloring of the page.

Javascript

Javascript is a programming language that was designed for and lives in the browser. It is designed to maniuplate both HTML and CSS - so that you can actually program all the aspects of the content of your page and the style it has.

HTML, CSS, and Javascript

are all designed to work together. CSS works by targeting HTML elements and writing the style aspects of it. With Javascript you can add, change or remove both CSS and HTMl.
